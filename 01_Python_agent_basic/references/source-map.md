# Source Map and Learning Adaptation

## Primary reference

- Original learning repository: `shendesuchit/learning-agentic-ai`
- Source module: `01_Python_agent_basic`

## How this edition transforms the source

| Learning concept | Teaching artifact |
|---|---|
| Python function as a tool | `main.py` section 1 and lesson 2 |
| Tool schema | `main.py` section 2 and tool-calling visual |
| Model tool selection | Lesson 3 |
| Tool execution | `execute_tool_call()` |
| Agentic loop | `run_agent()` and complete-loop visual |
| Conversation state | Lesson 5 and message-history visual |

The edition keeps the project intentionally small, uses fixed weather data for repeatable
learning, and adds an offline model adapter so the complete control flow can be studied without
credentials.
