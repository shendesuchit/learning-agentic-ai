# Notebook Page: Agent Basics

> **Agent = model decision + application-controlled action**
>
> - Model: understands the request
> - Tool schema: explains available actions
> - Python function: performs the action
> - Messages: carry context and results
> - Agent loop: coordinates everything
>
> Memory trick: **Decide → Do → Describe**
>
> The model decides. Python does. The model describes the result.
