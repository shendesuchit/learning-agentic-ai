# Notebook Page: Agentic Loop

> 1. User asks
> 2. Model decides
> 3. Python executes when requested
> 4. Tool result joins the messages
> 5. Model answers using the new result
>
> Biggest “aha!”:
>
> **After a tool runs, call the model again.**
>
> A tool returns data. The final model call turns that data into a useful answer.
