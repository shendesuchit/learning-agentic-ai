# Sticky Notes: Remember These Concepts

> 🟨 **MODEL**  
> Chooses an answer or requests a tool. It does not run your Python function.

> 🟦 **SCHEMA**  
> The model-readable menu of tool names, descriptions, and inputs.

> 🟩 **REGISTRY**  
> Maps an allowed name to the real function Python may execute.

> 🟧 **MESSAGES**  
> The temporary working context sent with a model request.

> 🟥 **SECOND MODEL CALL**  
> Required so the model can see and interpret the tool result.
