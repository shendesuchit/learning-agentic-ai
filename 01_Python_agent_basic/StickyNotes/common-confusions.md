# Sticky Notes: Common Confusions

| Confusion | Correct understanding |
|---|---|
| “The model called Python.” | The model requested a call; the application executed it. |
| “The schema is the function.” | The schema only describes the function. |
| “The tool result is the final answer.” | It is data that normally returns to the model. |
| “The model remembered the result.” | The application included it in the next request. |
| “An agent requires a framework.” | A framework is optional; the loop can be plain Python. |
