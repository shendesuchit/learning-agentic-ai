# Agent Fundamentals — Sticky Notes

> 🧠 **Brain is a metaphor.** The LLM generates text or a structured request; it does not run your code.

> 🗂️ **Memory is managed state.** A message list gives continuity only while the application retains and resends it.

> 🧰 **Tool schema ≠ tool.** The schema describes a capability. The registry maps it to executable code.

> 🛂 **Model proposes; application authorizes.** Validate names, arguments, permissions, and policies before execution.

> 🔁 **Tool results must return to the model.** Raw data becomes a contextual user answer only after the next model call.

> 🧯 **Max turns is not full safety.** Add timeouts, budgets, retries, logging, and side-effect controls.

> 🧭 **Workflow is not agentic by default.** Workflows follow fixed steps; agents choose among approved next actions.
